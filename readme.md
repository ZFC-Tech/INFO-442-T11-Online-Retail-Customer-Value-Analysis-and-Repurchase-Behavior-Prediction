1. Project Overview

This project aims to apply data science techniques to deeply analyze transactional data from a multinational online retailer. The primary goal is not only to describe historical purchasing patterns but also to build predictive models to identify key characteristics of high-value customers and predict the likelihood of new customers making repeat purchases. Starting with fundamental exploratory data analysis, we will establish customer RFM (Recency, Frequency, Monetary) profiles. Ultimately, we will build and compare different machine learning models to move from descriptive analysis to predictive insights, providing data-driven decision support for refined customer operations.

2. Dataset

•   Name: Online Retail Dataset

•   Source: Kaggle Platform, containing real transaction records from a UK-based online retailer between 2010-2011.

•   Data Scale & Features:

    ◦   Original Size: Over 540,000 transaction records.

    ◦   Number of Features: Includes 8 core features such as InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, and Country, meeting the requirement of at least 8 features. Key analytical variables like TotalPrice, Recency, Frequency, and Monetary will be derived from these.

•   Ethics & Applicability: This dataset consists of public, anonymized transaction data containing no personal sensitive information, adhering to research ethics. It avoids overly simplistic educational datasets like Titanic or Iris, offering a real business context and rich analytical dimensions.

3. Data Science Question with Scientific and Business Value

Core Question: Based on a customer's initial transaction behavior data, how can we effectively predict their future probability of becoming a "repeat-purchase customer"? Which factors most significantly influence customer repurchase behavior?

•   Scientific Value:

    1.  Methodology Application: This project will systematically apply and compare multiple machine learning classification algorithms (e.g., Logistic Regression, Decision Trees, Random Forest, Gradient Boosting Machines) to solve the classic classification problem of customer repurchase prediction. We will explore the impact of feature engineering on model performance.
    2.  Hypothesis Testing: We will use rigorous statistical tests to validate patterns identified in preliminary analysis, ensuring robust conclusions and elevating the project beyond descriptive analysis to empirical research.

•   Business Value:

    1.  Increase Customer Lifetime Value: Accurately predicting repurchase likelihood allows companies to prioritize marketing resources towards high-potential new customers, significantly improving marketing ROI. Preliminary analysis shows repeat customers spend 7x more on average than new customers, validating this direction's value.
    2.  Enable Precise Customer Segmentation and Intervention: By identifying key factors influencing repurchase, companies can develop differentiated customer engagement strategies.
    3.  Optimize Inventory and Operations: Predicting repurchase behavior aids in more accurate demand forecasting and inventory management.
